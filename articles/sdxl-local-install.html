Stable Diffusion XL本地安装教程：6GB显卡从零部署全流程

📅 2026-05-27🏷️ AI绘画

SDXL硬件要求

配置最低推荐理想
显存4 GB8-12 GB16 GB+
系统内存16 GB32 GB64 GB
存储20 GB50 GB SSD100 GB+ SSD

4GB显存勉强可运行（需--medvram参数），6GB OK，12GB体验最佳（1024×1024约20秒/张）。

安装步骤（Windows）

1. 安装Python 3.10

从python.org下载Python 3.10.11（不要用3.11+，部分依赖不兼容）。安装时勾选"Add to PATH"。

2. 安装Git

从git-scm.com下载安装。

3. 下载Automatic1111 WebUI

git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git
cd stable-diffusion-webui

4. 下载SDXL模型

从Hugging Face或Civitai下载sd_xl_base_1.0.safetensors，放入models/Stable-diffusion/目录。

5. 首次启动

webui-user.bat

首次启动会自动安装依赖（约10-20分钟）。之后浏览器打开 http://127.0.0.1:7860。

低显存优化

编辑webui-user.bat加入启动参数：

set COMMANDLINE_ARGS=--medvram --opt-sdp-attention

--medvram将模型分块加载，降低显存占用。6GB显卡加此参数可流畅运行SDXL。

Fooocus——更简单的选择

如果觉得WebUI太复杂，试试Fooocus——专为SDXL设计的简化界面，安装更简单，默认效果更好。从GitHub下载即可。
