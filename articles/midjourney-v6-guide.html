核心参数详解

--ar（宽高比）

V6.1支持最大14:1的极端比例：

--ar 16:9  # 横屏桌面壁纸
--ar 9:16  # 竖屏手机壁纸
--ar 1:1   # 正方形（默认）

--stylize / --s（风格化程度）

值域0-1000，默认100。越高越艺术化，越低越忠实提示词：

--s 0    # 写实
--s 250  # 适度艺术化
--s 1000 # 最大风格化

--chaos / --c（多样性）

值域0-100。控制4张图差异程度。0=风格接近，100=差异巨大。

其他重要参数

  • --iw（图片权重）：0-3，控制参考图影响力
  • --seed：固定随机种子，复现同一风格
  • --no：排除元素，如 --no people, text
  • --tile：生成无缝图案
  • --weird：0-3000，引入超现实元素

高级功能

--sref（风格参考）

/imagine prompt:a city at night --sref [图片URL] --sw 800

--cref（角色参考）

上传角色图保持同一形象。--cw控制一致性强度（0-100）。

提示词公式

最有效结构：主体 + 场景 + 风格 + 光线 + 参数

a young woman in a cyberpunk market, neon lights,
cinematic lighting, photorealistic, 8k,
--ar 16:9 --s 250 --v 6.1
